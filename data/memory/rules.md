# 自己生成ルール

このファイルはMetaAgentが経験から学んだルールを記録します。
ルールは定期的に検証され、有効性が確認されないものは淘汰されます。

## フォーマット

```
### [RULE-XXX] ルール名
- 生成日: YYYY-MM-DD
- 最終検証: YYYY-MM-DD
- 信頼度: high/medium/low
- 適用回数: N回
- 成功率: XX%
- 状態: active/under_review/deprecated

**ルール内容:**
〜のとき、〜する（または〜しない）

**生成の経緯:**
どのような経験からこのルールが生まれたか

**検証履歴:**
- YYYY-MM-DD: 検証結果の概要
```
